# Metrocuadrado Property Listings Dataset Sample

<h2>3,386개 레코드로 구성된 샘플 데이터셋</h2>

<a href="https://brightdata.co.kr">

이 **Metrocuadrado Property Listings Dataset Sample**에는 **3,386개 레코드**가 포함되어 있으며 **Bright Data API**를 사용하여 추출되었습니다.

## Data Points

| Field Name | Description |
|------------|-------------|
| `url` | Metrocuadrado 웹사이트의 특정 부동산 매물로 바로 연결되는 직접 URL입니다. |
| `ID` | 각 부동산 매물에 부여되는 고유 식별자입니다. |
| `Precio` | 매물의 표시 가격으로, 판매 또는 임대 기준이며 현지 통화로 표기됩니다. |
| `Habitaciones` | 해당 부동산에서 이용 가능한 침실의 총 개수입니다. |
| `Banos` | 해당 부동산에서 이용 가능한 욕실의 총 개수입니다. |
| `Dimension_propiedad` | 제곱미터 기준 부동산의 총 건축 면적입니다. |
| `Dimension_terreno` | 제곱미터 기준 부동산의 총 토지 면적입니다. |
| `Comuna_Ciudad` | 부동산이 위치한 도시 또는 행정 구역입니다. |
| `Type1` | 'Apartamento' 또는 'Casa lote'와 같은 부동산 유형입니다. |
| `Venta_Arriendo` | 부동산이 판매인지 임대인지 나타내는 거래 유형입니다. |
| `Nuevo_Usado` | 부동산이 신규('Nuevo')인지 중고/기존('Usado')인지 여부를 나타냅니다. |
| `Region` | 부동산이 위치한 도시 내의 특정 지역 또는 동네입니다. |
| `Estrato` | 부동산 위치의 사회경제적 계층 분류입니다. |
| `Parqueaderos` | 부동산에 포함된 주차 공간의 개수입니다. |
| `Year` | 부동산의 연식으로, 일반적으로 연수 범주로 분류됩니다. |
| `Descripcion` | 부동산의 특징 및 편의시설에 대한 상세 설명입니다. |
| `Seller` | 부동산을 판매 또는 임대하는 개인 또는 회사의 이름입니다. |
| `Tipo_de_vendedor` | 'Owner' 또는 'Agency'와 같은 판매자 유형입니다. |
| `Phone1` | 부동산 문의를 위한 기본 연락처 전화번호입니다. |
| `Phone2` | 추가 문의를 위한 보조 연락처 전화번호입니다. |
| `Email` | 부동산 문의를 위한 연락처 이메일 주소입니다. |
| `Imagen` | 부동산 이미지로 연결되는 URL입니다. |
| `URL` | 부동산 매물로 바로 연결되는 직접 URL의 중복 필드입니다. |
| `company_id` | 해당 부동산을 등록한 회사의 식별자입니다. |


## Use Cases

### 1. Real Estate Market Analysis
이 데이터셋을 활용하여 다양한 지역의 시장 동향, 가격 책정 전략, 투자 기회를 파악할 수 있습니다.

### 2. Competitive Analysis for Real Estate Agents
에이전트는 이 데이터를 활용하여 부동산 특징과 가격을 비교함으로써 시장에서의 경쟁 우위를 강화할 수 있습니다.

### 3. Urban Development Research
연구자는 도시화 패턴과 주거 수요를 연구하여, 도시 계획자가 정보에 기반한 의사결정을 내리는 데 도움을 줄 수 있습니다.
